4.1. 3-D Secure version determinationÂ¶

There are currently three 3-D Secure versions, 1.0.2, 2.1.0 and 2.2.0.

  1. If the preauth call returns a BIN not enrolled error, continue with 3DSv1.

  2. If acsInfoInd (2.1.0, 2.2.0) is included, this indicates if ACS supports challenges or only attempts.

    To select between 2.1.0 and 2.2.0, use the highest supported version between dsEndProtocolVersion and acsEndProtocolVersion, and the versions your implementation supports.
